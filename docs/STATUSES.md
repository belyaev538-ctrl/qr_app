# Статусы и типы полей из базы данных

Справочник полей коллекций `orders` и `order_items`, их типов и допустимых значений.

---

## Заказ (orders)

### Все поля заказа

| Поле в базе          | Тип поля    | Описание                                          |
|----------------------|-------------|---------------------------------------------------|
| `id`                 | string      | ID документа                                      |
| `order_number`       | string      | Номер заказа (например A-11)                      |
| `store_id`           | string      | ID магазина                                       |
| `order_type`         | string      | Вид получения (см. таблицу ниже)                  |
| `order_total`        | number      | Сумма заказа, руб.                                |
| `items_total`        | number      | Общее кол-во единиц товаров                       |
| `items_collected`    | number      | Сколько единиц уже собрано                        |
| `priority_status`    | string      | Приоритет заказа (см. таблицу ниже)               |
| `queue_position`     | number      | Позиция в очереди                                 |
| `status`             | string      | Статус заказа (см. таблицу ниже)                  |
| `collect_until`      | timestamp   | Собрать до (дедлайн)                              |
| `created_at`         | timestamp   | Дата создания                                     |
| `picker_id`          | string \| null | ID сборщика (кто собирает)                     |
| `picking_started_at` | timestamp \| null | Когда начали сборку                           |
| `picking_finished_at`| timestamp \| null | Когда закончили сборку                        |
| `comment`            | string \| null | Комментарий к заказу                           |

### Вид получения заказа (order_type)

| Значение   | Расшифровка        |
|------------|--------------------|
| `delivery` | Доставка           |
| `pickup`   | Самовывоз (Заберут)|

### Статус заказа (status)

| Значение       | Расшифровка       |
|----------------|-------------------|
| `not_started`  | Еще не собирается |
| `collecting`   | Собирается        |
| `collected`    | Заказ собран      |
| `out_of_stock` | Закончился товар  |
| `cancelled`    | Заказ отменён     |
| `done`         | Выполнен          |

### Приоритет заказа (priority_status)

| Значение        | Расшифровка   |
|-----------------|---------------|
| `urgent`        | Срочный       |
| `collect_now`   | Пора собирать |
| `can_start`     | Можно начать  |
| `plenty_of_time`| Время есть    |

---

## Товар в заказе (order_items)

### Все поля товара

| Поле в базе         | Тип поля | Описание                           |
|---------------------|----------|------------------------------------|
| `id`                | string   | ID документа                       |
| `order_id`          | string   | ID заказа                          |
| `product_id`        | string   | ID товара/продукта                 |
| `product_image`     | string   | URL картинки                       |
| `product_name`      | string   | Название товара                    |
| `quantity`          | number   | Сколько нужно собрать              |
| `quantity_collected`| number   | Сколько собрано                    |
| `unit`              | string   | Единица (л, кг, шт)                |
| `status`            | string   | Статус товара (см. таблицу ниже)   |

### Статус товара (status)

| Значение       | Расшифровка       |
|----------------|--------------------|
| `pending`      | Ожидает обработки  |
| `collected`    | Собран             |
| `out_of_stock` | Нет в наличии      |
